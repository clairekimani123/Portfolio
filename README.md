# Claire Kimani — Developer Portfolio 🚀

> Personal portfolio showcasing full-stack development projects, skills, and experience.

**Live:** [claire-portfolio.vercel.app](https://portfolio-nine-xi-fc7wggw8ye.vercel.app)

---

## About

A single-page portfolio website built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies. Professional navy/gold design with smooth scroll animations, a custom cursor, and fully responsive layout.

---

## Features

- ✨ Custom animated cursor with lag-follow ring effect
- 🎞️ Scroll-triggered reveal animations (IntersectionObserver)
- 📱 Fully responsive — mobile, tablet, desktop
- 🧭 Sticky nav with active section highlighting
- 💼 Projects section with live links and "Deploying Soon" badges
- 📬 Contact section with form and direct contact details

## Deployment

Deployed on **Vercel** — push to `main` to auto-deploy.

```bash
# To run locally — just open index.html in a browser
# No build step required
open index.html
```

---

## Customisation

To update project links once deployed:
1. Find the `project-links` section for the relevant project in `index.html`
2. Replace the `href="#"` or "Deploying Soon" badge with the live URL
3. Push to GitHub — Vercel auto-deploys
